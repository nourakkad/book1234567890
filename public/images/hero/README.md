# Hero / Parchment images

Put your three design images in this folder. Names must match exactly (lowercase, with hyphen):

| Save as          | Description |
|------------------|-------------|
| parchment-1.png OR parchment-1.jpg | Dark brown torn parchment on black |
| parchment-2.png OR parchment-2.jpg | Light beige with black bar at bottom |
| parchment-3.png OR parchment-3.jpg | Light brown parchment with curled corner |

- The app checks for .png first, then .jpg. Use either extension.
- Path: `public/images/hero/` (this folder).
- These names also work: 1.png, 2.png, 3.png (and .jpg). Preferred: parchment-1, parchment-2, parchment-3.
